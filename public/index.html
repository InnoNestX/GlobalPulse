Features

📨

Multi-Platform Push

Push to Feishu, WeChat Official Account, WeChat Clawbot, and Telegram simultaneously.

⏰

Scheduled Briefings

Configurable push times with timezone support. Market-calendar-aware scheduling.

🎨

Admin Web UI

Password-protected admin interface backed by Cloudflare Workers KV. No code required.

📅

Market Calendars

Everyday, A-share, US stock, and crypto trading day checks with holiday lookups.

👁️

Message Preview

Preview exactly what each platform will receive before sending.

🔗

Event Webhooks

Integrate with GitHub Actions and Cloudflare events for automated workflows.

API Endpoints

POST /v1/messages

Direct push message to configured targets. Requires API_TOKEN in Authorization header.

AuthorizationRequired
Bearer token: Bearer YOUR_API_TOKEN
header
contentRequired
Message content (markdown or text)
string
targetsOptional
Comma-separated: feishu, wechat_official_account, wechat_clawbot, telegram
string
curl -X POST https://globalpulse.workers.dev/v1/messages \ -H "Authorization: Bearer YOUR_API_TOKEN" \ -H "Content-Type: application/json" \ -d '{"content": "Hello World", "targets": "feishu,telegram"}'
POST /v1/events/github-actions

Receive GitHub Actions workflow run events. Triggers briefing based on event type.

curl -X POST https://globalpulse.workers.dev/v1/events/github-actions \ -H "Authorization: Bearer YOUR_API_TOKEN" \ -H "Content-Type: application/json" \ -d '{"action": "completed", "workflow": "ci"}'
POST /v1/events/cloudflare

Receive Cloudflare Workers KV events, zone changes, or analytics alerts.

curl -X POST https://globalpulse.workers.dev/v1/events/cloudflare \ -H "Authorization: Bearer YOUR_API_TOKEN" \ -H "Content-Type: application/json" \ -d '{"type": "kv", "key": "config"}'

Deployment

1

Clone and Install

Clone the repo and install dependencies.

git clone https://github.com/InnoNestX/GlobalPulse.git cd GlobalPulse npm install
2

Configure

Copy example files and add your secrets.

cp .dev.vars.example .dev.vars cp wrangler.example.jsonc wrangler.jsonc
3

Run Locally

Start dev server and access the admin UI.

npm run dev # Open http://localhost:8787/admin
4

Deploy to Cloudflare

Deploy with Wrangler using your Cloudflare credentials.

npm run deploy
