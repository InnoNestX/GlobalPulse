自定义变量 ​

通过管理后台设置的每任务变量：

  • - 配置的应用名
  • - 任务标识符
  • - a_share、us_stock、crypto 等

技巧 ​

  1. 保持简洁 - 渠道有消息长度限制
  2. 适度使用 emoji - 不同平台渲染不同
  3. 用预览测试 - 启用前先用预览确认
  4. 时区明确 - 底部包含时区便于理解
