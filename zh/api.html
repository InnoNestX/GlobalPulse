管理后台 API ​

所有管理接口需要通过 ADMIN_PASSWORD header 或 cookie 认证。

登录 ​

POST /api/admin/login
1

请求：

json
{ "password": "your-admin-password" }
1

响应：

json
{ "success": true }
1

}
